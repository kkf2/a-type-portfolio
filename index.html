福岡県在住。障害者手帳を持ちながら、Web制作とコンテンツ作成に取り組んでいます。

個人ブログの運営を通じて、情報を整理し、分かりやすく伝えることを大切にしてきました。

できること

code

HTML / CSSでのページ作成・修正

テキストの差し替え、レイアウトの調整、色やフォントの変更など、既存サイトの更新・修正作業に対応できます。

edit_note

ブログ記事の執筆

個人ブログの運営を通じて、構成を考えてから書く記事作成の経験があります。見出しや段落の整理を意識したライティングが得意です。

palette

アイキャッチ・バナーなどの画像作成

ブログのアイキャッチ画像、ヘッダー画像、アイコン、ファビコンなどの作成経験があります。テキスト配置、エフェクトや図形の追加、透明度・シャドウの調整といった基本的な画像編集を行ってきました。

table_chart

情報の整理・構成設計

Googleスプレッドシートを使ったデータの整理や、ドキュメントでの情報のまとめ作業ができます。要件や手順を分かりやすく構造化するのが得意です。
